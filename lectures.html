• Getting started with SVG for Web Design

    With Gary Simon, August 30, 2014 at 11:00 a.m.

    In this lecture, Gary Simon will teach you how to use SVG (Scalable Vector Graphics) in your web design projects. You’ll learn how to design and animate various web elements including a loader, animated icons, and line animations. Alongside gaining skills in CSS, JavaScript and HTML, you’ll also learn how to use various design applications including Adobe Illustrator, Adobe Photoshop and also Adobe Edge Animate. And lastly, you’ll learn how to make all these SVG animations responsive.

  • Accessing External APIs Using AngularJS's Services

    With Maciej Sopyto, September 11, 2014 at 9:00 a.m.

    Aside from ability to extend HTML easily AngularJS also offers a simple way to interact with external APIs. In this tutorial, I will show you how to use Services to access GitHub's API and create a simple repository browser.

  • Bootstrap 3 for Web Design

    With Adi Purdila, September 18, 2014 at 11:00 a.m.

    Bootstrap is the most popular HTML, CSS, and JS framework for developing responsive, mobile first projects on the web. In this lecture you'll learn everything you need to know in order to get started with the very popular Twitter Bootstrap framework.
