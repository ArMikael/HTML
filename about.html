Geenio - web-based online tool for interactive training courses creation.

Easy to Use LMS - Geenio has simple and yet powerful learning management system that allows you to manage learning process in a relaxed and very comfortable manner. It allows you to manage groups and users in an comfortable way. Get detailed reports and statistics on users performance and course efficiency.

Innovative Visual Editor - Create interactive professional courses right in your web browser without a single line of code. Use simple widgets to add media on pages: text, image, video, sound, shapes and more. With Geenio you can create tests. Choose between different customizable question types. Attach images and videos to make your test more informative and interesting. With its responsive interface, Geenio is working perfect on all possible devices and screen resolutions.

The Pathboard - Pathboard is a non-linear course planner. Use triggers to determine transition between elements. Transform your lecture into a game!
